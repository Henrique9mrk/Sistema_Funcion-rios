# 📊 Sistema de Funcionários

Sistema web desenvolvido em Python utilizando Streamlit para gerenciamento de funcionários.

## 🚀 Funcionalidades

- Cadastro de funcionários
- Edição de dados
- Exclusão
- Dashboard com métricas
- Busca por nome

## 🛠️ Tecnologias

- Python
- Streamlit
- SQLite
- Pandas

## ▶️ Como executar

```bash
pip install -r requirements.txt
python -m streamlit run app.py
